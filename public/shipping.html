Automated logistics

From checkout to courier pickup—automatically.

When the automation server and live courier credentials are configured, an order can be sent to the logistics provider, assigned an AWB, scheduled for pickup and tracked from the same ApnaFinds website.

Check courier serviceability

Enter a customer PIN code to view available demo or live courier options.

1. Order received The checkout sends the purchased products and delivery address to the server.
2. Courier selected The server checks PIN-code serviceability and chooses a configured or recommended courier.
3. AWB and pickup The courier AWB is assigned and pickup is requested from the seller location.
4. Customer tracking Courier updates flow into Track Order and the AI helper.

Delivery, cancellation and return rules

Cancellation: customers can normally self-cancel while an order is Order placed, Confirmed or Packed. Once pickup or transit begins, courier cancellation may no longer be possible.

Returns and replacements: a reverse pickup can be created after the order is delivered and the return request passes the store’s review rules.

Refunds: the website records and tracks refund requests. Moving actual money requires the original payment gateway or a verified payout process. Cash on Delivery refunds require customer payout details and admin approval.
