# Install First

Use only the extracted complete folder.

1. Install Node.js 18+.
2. Double-click `START-APNAFINDS.bat`.
3. Sign in at `http://localhost:3000/admin-login.html`.
4. Use `admin@apnafinds.local` and `ApnaFinds@123` for the first local test.
5. Change the admin credentials and secrets in `.env` before public hosting.

Do not open the HTML files directly and do not use Live Server.
